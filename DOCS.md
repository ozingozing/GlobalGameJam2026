##  아레나 진행 방식
기본모드:  30분 (배틀 파트) -> 5분 휴식 (상점파트) -> 30분 (배틀파트) -> 5분 휴식 (상점파트)... 

배틀파트: 적 무한 랜덤 리스폰, 골드 드랍 

즉 성장 컨텐츠 = 상점 활용 가능한 "골드" 재화 (경험치가 아님)

(일단 지금은 기본 모드만 구현!!)

### Pause 패널 (ESC 키)
아레나 씬에서 ESC 누를 시 Pause 패널 팝업
- Continue: 패널 닫기 (ESC 다시 눌러도 닫힘)
- Setting: 메인메뉴와 동일한 세팅 패널 팝업 (DontDestroyOnLoad로 데이터 연동)
- Exit: "메인 메뉴로 가시겠습니까?" 확인 패널 팝업

### 사망 패널
죽었을 때 게임 일시정지 + "죽었습니다!" 패널 팝업
- Retry 버튼
- Home 버튼

중앙: 마인크래프트식 인벤토리 UI 

배틀 파드 클리어 시 canvas group으로 "클리어!" 등장 후 사라짐

## 캐릭터 표정 (가면)
표정A: 공격력 20% 상승
표정B: 데미지 피격 20% 감소
표정C: 캐릭터 이동속도 20% 상승
표정D: 위의 효과들 5%씩 모두 상승

(효과 구현까진 아직은 안하셔도 됩니다!)

## 공격 시스템 (표정 매칭)
~~무기 시스템은 삭제됨~~

캐릭터 표정 <-> 에너미 표정 매칭으로 데미지 여부 판정
- 표정A 캐릭터 → 표정A 에너미에게 데미지 가능
- 표정B 캐릭터 → 표정B 에너미에게 데미지 가능
- 표정C 캐릭터 → 표정C 에너미에게 데미지 가능
- 표정D 캐릭터 → 표정D 에너미에게 데미지 가능

대응되는 표정끼리만 공격 가능

## 1차 스코프 아트 에셋

### 플레이어 & 에너미
- 표정A, 표정B, 표정C, 표정D
- 에너미는 색깔 바리에이션 x4 (색깔1~4)

### 모션 (4방향, 좌우상하반전 활용)
- 평상: 2프레임 (스프라이트 낮아짐 -> 높아짐 반복)
- 걷기: 3프레임 (왼발 -> 중앙 -> 오른발)
- 피격: 2프레임 (평상 or 걷기 기반)
- 사망: 3프레임 (팔벌림 -> 비스듬 -> 완전히 누움)
- 공격: 2프레임 (평상 or 걷기 -> 팔모으기/팔뻗기)

## 적
4 바리에이션의 기본 적
